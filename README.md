## Quick Start Commands

```
<!-- Start Command -->
yarn <framework>:start

<!-- Build Command -->
yarn <framework>:build

<!-- Build Playground Command -->
yarn <framework>:build:playground

<!-- Build All -->
yarn build

<!-- Lint Command -->
yarn <framework>:lint

<!-- Lint All -->
yarn lint

<!-- Typecheck Command -->
yarn <framework>:typecheck

<!-- Typecheck All -->
yarn typecheck

<!-- Test Command -->
yarn <framework>:test

<!-- Test All -->
yarn test
```
